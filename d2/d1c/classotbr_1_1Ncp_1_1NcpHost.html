◆ Leave()

void otbr::Ncp::NcpHost::Leave ( const AsyncResultReceiver &  aRecevier)
overridevirtual

This method instructs the device to leave the current network gracefully.

  1. If there is already an ongoing 'Leave' operation, no action will be taken and aReceiver will be called after the previous request is completed. The previous aReceiver will also be called.
  2. If this device is not in disabled state, OTBR sends Address Release Notification (i.e. ADDR_REL.ntf) to gracefully detach from the current network and it takes 1 second to finish.
  3. Then Operational Dataset will be removed from persistent storage.
  4. If everything goes fine, aReceiver will be invoked with OT_ERROR_NONE. Otherwise, other errors will be passed to aReceiver when the error happens.
Parameters
[in]aReceiverA receiver to get the async result of this operation.

Implements otbr::Ncp::ThreadHost.
